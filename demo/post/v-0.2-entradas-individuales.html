v 0.2 entradas individuales

para poder ver el contenido de las entradas hay que crearles un single.php

single.php :


        <main>
        <?php  if ( have_posts() ) { while ( have_posts() ) { the_post(); ?>
        <post>
          <h1><?php the_title(); ?></h1>
          <?php the_content(); ?>
        </post>
        <?php } } ?>
        </main>

y para separarlo mejor de index ahora las entradas de index van a ser links con h2 en lugar de h1

index.php :


        <main>
        <?php  if ( have_posts() ) { while ( have_posts() ) { the_post(); ?>
          <post>
            <h2><a href="<?php the_permalink(); ?>"><?php the_title(); ?></a></h2>
          </post>
        <?php } } ?>
        </main>
